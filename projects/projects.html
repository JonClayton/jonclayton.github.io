My first shareable project was a working Bingo game in Ruby. You can find the code here. Please feel free to copy it and paste it into a Ruby file to run and test it out.

My second shareable project is a game. I'm working on a checkers game. You can see it here. I am pretty excited. It runs for two players, applying all the checkers rules. Further improvements I aim to make before the deadline for turning it in are:

  • Text instructions in the box to the right that are dynamic.
  • Indicator for which checker you've clicked on to move.
  • Let you put in player names (and uses them of course!)

Things I plan to take a shot at but am not sure I can get them to work:

  • Cleaning up the CSS so it scales rather than goes wacky when you reduce the window size.
  • Implement at least a little AI so it can be played against the computer. I would be able to leverage that logic to highlight the moveable pieces for human player(s) and once they select one, show the places they can be moved.
  • Make some general improvements to attractiveness.
